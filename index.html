HTML Escaper

Was bedeutet "Escapen"?

Wenn du HTML-Code auf einer Webseite zeigen willst, interpretiert der Browser ihn normalerweise als echtes HTML. Um das zu verhindern, ersetzt du bestimmte Zeichen durch ihre HTML-Entities:

  • < wird zu &lt;
  • > wird zu &gt;
  • & wird zu &amp;
  • " wird zu &quot;
  • ' wird zu &#39;

So wird aus <div> der Text &lt;div&gt;, der dann als lesbarer Code angezeigt wird.

Je nach Umfang kann das sehr zeitaufwendig werden, daher gibt es hier dieses kleine Tool.

Escape & Unescape

Das Tool funktioniert in beide Richtungen:

→ Escapen

Konvertiert HTML-Code in sichtbaren Text:

<div> → &lt;div&gt;

← Unescapen

Konvertiert escaped Code zurück in HTML:

&lt;div&gt; → <div>

Pro-Tipp: Mehrfach-Escapen

Du kannst Code auch mehrfach escapen - das ist super nützlich, wenn du zeigen willst, wie Escapen funktioniert!

Beispiel:

  1. Start: <div>
  2. 1× Escapen: &lt;div&gt;
  3. 2× Escapen: &amp;lt;div&amp;gt;
  4. 1× Unescapen: &lt;div&gt;
  5. 2× Unescapen: <div>

So kannst du in Tutorials zeigen, dass &lt; zu < wird, ohne dass der Browser es direkt umsetzt!
